Create a smart radio station with The Echo Nest

Using The Echo Nest API, grab the user's currently playing track, or their top artist. If the user clicks "I like it" button in the first 30 seconds, add the song to a playlist. After 30 seconds, if the user hasn't liked the song, skip it.

The Javascript

var sp = getSpotifyApi(1);
var models = sp.require('sp://import/scripts/api/models');
var views = sp.require('sp://import/scripts/api/views');

The HTML

<div id="player"></div>

The CSS
